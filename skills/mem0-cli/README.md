# mem0-cli Skill

Teaches AI agents how to use the [mem0 CLI](../../mem0-cli/) for persistent memory — storing learnings, retrieving past context, and building knowledge across sessions.

## What this skill covers

- When and why to use memory (before tasks, after fixes, on confirmed decisions)
- All CLI commands with examples and expected output
- JSON output parsing for scripting and automation
- Agent-specific patterns (retrieve-then-act, store-after-confirming)
- Guidelines on what's worth storing vs. what isn't

## Difference from the `mem0` skill

| | `mem0` skill | `mem0-cli` skill |
|--|-------------|-----------------|
| **Teaches** | How to write code using the mem0 SDK | How to use mem0 from the terminal |
| **For** | Building apps with persistent memory | Agents and scripts that need memory |
| **Auth** | Requires pre-configured API key | Can self-authenticate via CLI |
| **Language** | Python, TypeScript | Shell (any platform) |
